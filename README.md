# EcoMonitor AI (سامانه هوشمند مدیریت و پایش آلاینده‌های صنعتی)

سامانه هوشمند **EcoMonitor AI** یک پلتفرم فرانت‌اند فوق‌پیشرفته، واکنش‌گرا و مبتنی بر هوش مصنوعی مولد است که برای پایش لحظه‌ای، عیب‌یابی فنی مشعل بویلرها و انطباق آلاینده‌های خروجی دودکش با استانداردهای مدیریت زیست‌محیطی **ISO 14001** طراحی شده است. این سامانه به مهندسان محیط‌زیست و تاسیسات کمک می‌کند تا مقادیر گازهای احتراقی و ذرات معلق دودکش را ردیابی کرده و مغایرت‌های استاندارد را با راهکارهای مهندسی اصلاح کنند.

---

## 🌟 ویژگی‌های کلیدی (بر اساس اولویت و ارزش کاربردی)

### ۱. معماری هوش مصنوعی هیبریدی سه‌لایه (Hybrid AI Multi-Layer Router)
قلب تپنده سامانه پایش، پردازش هوشمند رویدادها با استفاده از استراتژی توزیع بار میان ابر و لبه است:
- **Cloud Gemini Engine**: استفاده از موتورهای نوین کلود با پردازشگرهای `gemini-3-flash` و `gemini-3-pro` برای تحلیل‌های چندبعدی پیشرفته، رولهای تفصیلی و استخراج راهکارهای نگهداری بویلر.
- **Offline Ollama Core**: امکان اتصال مستقیم به سرویس‌های هوش مصنوعی بومی مستقر در شبکه داخلی کارخانه (روی پورت پیش‌فرض `11434` با مدل‌های پیش‌فرض مانند `Llama3`).
- **WebAssembly Local Assistant**: یک هسته قوانین خبره محلی (بر پایه موتور Llama-FA) با عملکرد ۱۰۰٪ آفلاین و بدون نیاز به شبکه، جهت صدور فوری گزارش‌های فنی بدون خطر افشای اطلاعات صنعتی.

### ۲. پایش هوشمند انطباق استاندارد (ISO 14001 Combustion Compliance)
سنجش و تحلیل خودکار پارامترهای بحرانی دودکش به محض ورود داده‌ها:
- ارزیابی لحظه‌ای آلاینده‌های اصلی: مونوکسید کربن (CO)، اکسیدهای نیتروژن (NOx)، دی‌اکسید گوگرد (SO2)، ذرات معلق (PM)، دی‌اکسید کربن (CO2) و نسبت اکسیژن آزاد (O2).
- طبقه‌بندی هوشمند وضعیت فرآیند احتراق به سه سطح **نرمال (سبز)**، **هشدار مرزی (زرد)** و **بحرانی/مغایرت زیست‌محیطی (قرمز)** با محاسبه خودکار نسبت به حدود استاندارد مجاز.

### ۳. تصویرسازی پیشرفته داده‌ها و تحلیل روند (Interactive Recharts Analytics)
ارائه نمودارهای تعاملی مدرن برای مانیتورینگ روندها:
- **نمودار گازهای احتراقی**: نمایش همزمان مونوکسید کربن (CO) در کنار اکسیدهای نیتروژن (NOx).
- **نمودار ذرات معلق و گوگرد**: پایش غبار خروجی و دی‌اکسید گوگرد نسبت به عملکرد فیلترها.
- **روند تاریخی چندگانه**: استفاده از چارت‌های ترکیبی (Area، Bar و Line) توسعه یافته با کتابخانه Recharts جهت مقایسه مقادیر دوره‌های فصلی با خط قرمز استانداردهای زیست‌محیطی.

### ۴. دستیار هوشمند و چت‌بات کارشناسی (Interactive AI Maintenance Chatbot)
یک سیستم مبتنی بر گپ‌وگفت فنی که با تجهیز به دستورالعمل‌های سیستمی تخصصی (System Instructions) قادر است به سوالات پیچیده مهندسان بویلر در زمینه‌های عیب‌یابی، کالیبراسیون سنسورها، تنظیم دمپر سوخت و هوا، راندمان حرارتی و الزامات ممیزی پاسخ‌های فنی و دقیق ارائه دهد.

### ۵. مدیریت جامع داده‌های پایش (Dynamic Inventory & Data Logs)
- ابزار ورود مستقیم اطلاعات اندازه‌گیری فصلی برای بویلرها و اگزوزهای مختلف به همراه ثبت موقعیت جغرافیایی.
- امکان افزودن تجهیزات جدید به خط پایش.
- قابلیت پشتیبان‌گیری و درون‌ریزی داده‌ها (Import/Export JSON) برای همگام‌سازی آسان داده‌های بازرسان.

### ۶. شخصی‌سازی پویای رابط کاربری (Professional UX Theme Engine)
طراحی بصری خارق‌العاده بر مبنای استانداردهای مدرن فرانت‌اند:
- **پشتیبانی کامل از حالت تاریک و روشن (Dark / Light Mode)** با رنگ‌بندی‌های ایمن برای کار در شیفت‌های شبانه اتاق کنترل صنعتی.
- **پنج پالت رنگی غنی و جذاب**: شبیه‌سازی تم‌های صنعتی هوشمند با کارهای آبی تجاری، زمردی محیط‌زیستی، بنفش فناوری، سرخ اضطراری و کهربایی سنتی.
- **چندزبانه پیشرفته (i18n)**: تغییر راحت زبان برنامه بین فارسی (RTL کاملاً تراز شده با فونت خوانای وزیری) و انگلیسی (LTR تخصصی).

## 🛠️ تکنولوژی‌های مورد استفاده

- **فریم‌ورک اصلی**: React 19 به همراه زبان نوع‌دهی شده TypeScript جهت تضمین پایداری کد در مقیاس صنعتی.
- **ابزار ساخت**: Vite (سریع‌ترین رانر مدرن توسعه وب).
- **سیستم استایل‌دهی**: Tailwind CSS برای طراحی کاملاً واکنش‌گرا و پویا در نمایشگرهای صنعتی بزرگ، تبلت‌ها و گوشی‌های همراه.
- **نمودارها و گراف‌ها**: Recharts (بهینه‌شده برای رندرهای بدون تاخیر).
- **هوش مصنوعی**: کتابخانه رسمی `@google/genai` گوگل جهت برقراری ارتباط پایدار با مدل‌های هوش مصنوعی نسل جدید Google Cloud.
- **نمادها**: Lucide React.
- **ذخیره‌سازی اطلاعات**: LocalStorage (مرورگر کلاینت) برای نگه‌داری دائمی و آفلاین اطلاعات پایش کاربران به موازات قابلیت‌های اکسپورت/ایمپورت داده‌ها.

## 🚀 راهنمای راه‌اندازی سریع

برای راه‌اندازی پروژه بر روی سیستم لوکال خود، مراحل زیر را به ترتیب دنبال کنید:

### ۱. نصب وابستگی‌ها
ابتدا ترمینال را در پوشه ریشه پروژه باز کرده و دستور زیر را برای بارگیری وابستگی‌ها اجرا کنید:
```bash
npm install
```

### ۲. تنظیم متغیرهای محیطی
یک فایل به نام `.env` در محیط ریشه بسازید و کلید API اتصال به سرویس هوش مصنوعی گوگل را وارد کنید:
```env
API_KEY=your_gemini_api_key_here
```

### ۳. اجرای محیط توسعه (Development Server)
برای اجرای آزمایشی برنامه و مشاهده خروجی در مرورگر خود با آدرس `http://localhost:3000` دستور زیر را ارسال کنید:
```bash
npm run dev
```

### ۴. ساخت پیل نهایی (Production Build)
جهت ایجاد کدهای کامپایل‌شده نهایی با کارایی فوق‌العاده بالا جهت میزبانی تحت وب یا بسته‌بندی در قالب نرم‌افزارهای دسکتاپ و موبایل:
```bash
npm run build
```
کدهای ساخته‌شده در پوشه بهینه `dist/` قرار خواهند گرفت.

## 🌐 ایمنی و سازگاری هوشمند
سامانه مجهز به نسخه **SafeMode** اختصاصی است که در صورت قطع هماهنگی محیط Sandbox و برقراری مجدد وب‌سوکت در زمان کار، از لودشدن‌ها و فلش‌های ناگهانی فریم پیش‌نمایش در تداخل‌های محلی جلوگیری می‌کند. این قابلیت تجربه کاربری بی‌نقصی ارائه می‌دهد که همگام با نسخه‌های آینده وب، دسکتاپ (Electron/Tauri) و بسته‌بندی‌های موبایل (Capacitor) عملکرد ۱۰۰٪ بدون مشکلی خواهد داشت.
